TO INSTALL CHAT SERVER ON UBUNTU
1 - first install nodejs with command: sudo apt-get install nodejs
2 - install npm : sudo apt-get install npm
3 - install node-static: sudo npm install -g node-static
4 - install socket.io  : sudo npm install -g socket.io@0.9.17
5 - unzip the chat-server.zip, cd to it, then run: nodejs server.js


TO INSTALL CHAT SERVER ON MAC, use HOMEBREW
1 - first install nodejs with command: sudo apt-get install nodejs
2 - install npm : brew install npm
3 - install node-static in chat-server folder: npm install node-static
4 - install socket.io in chat-server folder : npm install socket.io@0.9.17
5 - unzip the chat-server.zip, cd to it, then run: node server.js
